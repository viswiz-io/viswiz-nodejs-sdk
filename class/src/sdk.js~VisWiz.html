    public buildFolder(build: object, folderPath: object, progressCallback: function): Promise source

    Creates a new build and uploads all images (*.png) found in a folder (scanned recursively)

    Params:

    NameTypeAttributeDescription
    build object
    build.branch string

    The branch name for this build
    build.projectID string

    The requested project ID
    build.name string

    The commit name for this build
    build.revision string

    The revision for this build
    folderPath object
    progressCallback function
    • optional

    called with parameters: (current, total)

    Return:

    Promise

    Example:

    await client.buildFolder({
  branch: 'master',
  projectID: 'mwwuciQG7ETAmKoyRHgkGg',
  name: 'New amazing changes',
  revision: '62388d1e81be184d4f255ca2354efef1e80fbfb8'
}, '/path/to/folder/with/images');
